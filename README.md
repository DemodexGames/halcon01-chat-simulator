# HALCON0.1 v8.3 — Chat Simulator

Proyecto web estilo WhatsApp creado por Demodex Games.

HALCON0.1 es un simulador de chat con personalidad agresiva, respuestas automáticas, audios, emojis, subida de archivos, modo autónomo, estados dinámicos y apariencia inspirada en WhatsApp.

## Características

- Interfaz tipo WhatsApp
- Foto de perfil personalizada
- Fondo de pantalla personalizado
- Respuestas automáticas variadas
- Anti repetición
- Estados dinámicos
  - en línea
  - escribiendo...
  - grabando audio...
- A veces deja en visto
- 10 audios locales
- Audio random por comando
- Audio por triggers de texto
- Audio por fotosarchivos
- Emojis con respuestas distintas
- Subida de imágenes y archivos
- Modo chino aleatorio
- Diseño listo para GitHub Pages

## Cómo usar localmente

1. Descarga o clona el repositorio.
2. Coloca todos los archivos en la misma carpeta.
3. Abre `index.html` en tu navegador.

## Publicar en GitHub Pages

1. Sube todos los archivos al repositorio.
2. Ve a Settings.
3. Entra a Pages.
4. En Source, elige
   - Deploy from branch
   - Branch main
   - Folder  (root)
5. Guarda y espera unos segundos.

Tu sitio quedará publicado en una URL tipo

`httpsTUUSUARIO.github.iohalcon01-chat-simulator`
